Iodine Timer Based Car

2022-05-01 Tags: project

Description

In this project, I helped a chemical engineering student with a car design they were working on for a competition.
They designed a Iodine based timer that would go from a dark liquid to clear one (or the reverse I don't remember at this time), a photresist sensor and LED were placed in parallel across the sides of the liquid container so that as the liquid became clear, the photoresitor would change value and we would activate a motor start signal.
The car would then start and stop based on the iodine timers transitions.

It was a really fun project and I enjoyed the chance to help out in this interdisciplinary work. Also this project was pre-chat GPT so I did all this work from scratch!

Design

Power:

  • 12V Battery powers Arduino and main circuit 
  • On/Off controlled by toggle switch in line with positive power line of battery 
  • Buck Converter ensures constant 12V to circuit, prevents fluctuations from affecting logic circuitry 

Circuit Components:

  • Photosensitive Circuitry: 
  • LED - Constant light source powered by Arduino 5V pin 
  • Photoresistor - Voltage, provided by Arduino 5V pin, is measured across photoresistor by Arduino pin A0 as an analog value (0-1024). The analog value we see is a product of the photoresistor and additional resistor used. Voltage reading increases when brightness high, voltage decreases when voltage low. Arduino pin 2 set HIGH (5V) when voltage reading is above threshold. Pin 2 set LOW(0V) when voltage reading below threshold. Threshold determined by testing voltage reading when iodine clock on/off. 

Motor Circuit:

  • Digital Switch - NMOS acts as digital switch. By adjusting the voltage between Gate and Source we can turn the switch On or Off. The pin 2 HIGH signal turns ON the NMOS and switch is closed, current passes through and powers motor. When pin 2 LOW, the NMOS is OFF and acts as open switch--no current passes through at this point.  
  • Example Circuit:
  • Motor - A 12V rated motor is powered as long as switch is closed. A "flywheel" diode is connected in parallel to the motor to prevent back emf from damaging transistor and other circuitry. Back emf is generated when motor stops or is reversed.
