Doodle-rs

2025-10-01 Tags: project

Description

Doodle-rs is a project I started while re-familiarizing myself with Rust.
Using an RPi Pico 2W with a small I2C OLED, it renders pixel drawings from a web interface in real-time.
Using networking and async tasks from the Rust Embassy framework, the Pico connects to my local WiFi with a static IP address.
The web app connects and establishes a WebSocket connection for low-latency data transfer.
Each pixel update sends a 3-byte message (x, y, state) to the Pico, which immediately updates the OLED display via an async display task.
